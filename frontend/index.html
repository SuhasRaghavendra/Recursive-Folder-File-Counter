Algorithm Theory

🔵

Recursive DFS

Uses the call stack implicitly. Risks overflow on extremely deep trees. Simple and elegant.

Space: O(h)
🟣

Iterative DFS

Heap-allocated explicit stack. Overflow-safe. Peak nodes = maximum directory depth.

Space: O(h)
🟢

BFS

Heap-allocated circular queue. Explores level-by-level. Peak nodes = widest level.

Space: O(w)
